# English Academy App

Esta es una aplicación web completa para la gestión de una academia de inglés ubicada en Villavicencio, Colombia.

## Características Principales

- **Gestión de estudiantes**: Registro, perfil y progreso de cada estudiante
- **Sistema de pagos**: Integración con ePayco para pagos en línea
- **IA para transcripción**: Transcripción automática entre español e inglés
- **Calendario de clases**: Programación y recordatorios automáticos

## Tecnologías Usadas

El proyecto utiliza Vue.js con Composition API para el frontend, Django REST Framework para el backend, y Docker para containerización.

## Desafíos

Uno de los mayores desafíos fue implementar la transcripción con IA en tiempo real, garantizando precisión tanto en español como en inglés.
