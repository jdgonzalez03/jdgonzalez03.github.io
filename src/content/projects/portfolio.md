# Portfolio Personal

Este mismo sitio web que estás visitando. Un portafolio personal moderno, rápido y con un diseño hermoso.

## Características

- Diseño inspirado en Dracula theme
- Animaciones suaves con GSAP
- Totalmente responsivo
- Tema claro/oscuro
- Carga de contenido desde markdown
- Optimizado para SEO

## Por qué Astro?

Elegí Astro por su excelente rendimiento y porque me permite escribir componentes de manera intuitiva usando la sintaxis que ya conocía de Vue y otros frameworks.
